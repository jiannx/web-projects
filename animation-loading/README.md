# 加载动画


### 文件目录
* dist 编译生成的js，css
* doc 文档目录
* src 源码
  * assets 静态资源
  * scss 样式源码
* test 测试页面

### 命令
* `npm install -g bower gulp`
* `npm install`
* `bower install`
* `gulp build` 式打包js
* `gulp watch` 监测文件并打包
* `gulp server` 挂起服务器
* `gulp sass` 生成css
* `gulp clean` 清除生成文件

###  注
* scss默认引入[bourbon](https://github.com/thoughtbot/bourbon)，[normalize](https://github.com/necolas/normalize.css)
